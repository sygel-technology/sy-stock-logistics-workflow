This module adds a "Responsible" field in stock moves, which shows the
responsible user of the related stock picking.

**Table of contents**

.. contents::
   :local:

Usage
=====

The "Responsible" field in stock moves is displayed in Inventory >
Reporting > Move Analysis, in both tree an form views.
