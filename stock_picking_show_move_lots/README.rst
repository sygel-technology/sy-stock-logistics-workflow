============================
Stock Picking Show Move Lots
============================

Show lots in the move lines of the stock picking form


Installation
============

To install this module, you need to:

#. Only install


Configuration
=============

To configure this module, you need to:

#. Only install


Usage
=====

To use this module, you need to:

#. Go to a stock picking form view. In the operations tab you will always see the "Lots/Serial Numbers" column
